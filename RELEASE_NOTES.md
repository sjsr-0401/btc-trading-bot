## v0.4 (2026-02-25)

### 버그 수정

- **오래된 크로스 진입 차단**
  EMA 크로스가 발생한 지 수 시간이 지난 후에도 봇이 해당 크로스를 기준으로 진입하던 문제 수정.
  크로스 발생 캔들 이후 가격이 2% 이상 이동한 경우 진입을 스킵 ("늦은 진입 차단").

- **봇 시작 시 오래된 크로스 감지 범위 축소**
  `PreAnalyze`(봇 시작 시 사전분석)의 lookback을 8봉(2시간)→3봉(45분)으로 줄여
  시작 직후 오래된 크로스로 즉시 진입하는 현상 방지.

- **코인 전환 버그 수정**
  스캐너에서 다른 코인 클릭 시 차트가 BTC로만 표시되던 문제 수정.
  `AvailableSymbols` 로드 수량을 스캐너 코인 수와 일치하도록 수정, 타이밍 레이스 제거.

- **포지션 종료 후 차트 라인 잔존 버그 수정**
  SL/TP가 체결되어 포지션이 종료된 후에도 진입가/SL/TP 라인이 차트에 남아있던 문제 수정.

- **차트 스크롤 시 SL/TP 라인 미업데이트 수정**
  차트를 스크롤·줌할 때 SL/TP 수평선이 Y축과 함께 업데이트되지 않던 문제 수정.
  `RectangularSection` 렌더링 버그를 우회하여 `LineSeries<ObservablePoint>`로 교체.

## v0.3 (2026-02-23)

### 버그 수정

- **외부 청산 미감지 버그 수정**
  바이낸스 서버에서 SL/TP 주문이 체결될 때 봇이 포지션 청산을 감지하지 못해 거래 통계(승률/손익)에 기록되지 않던 문제 수정.
  `_openPosType`, `_openEntryPrice`, `_openQty` 추적 필드 추가, 메인 루프에서 외부 청산 감지 블록 삽입.

- **자동전환 시 HttpClient dispose 레이스 컨디션 수정**
  스캐너가 더 좋은 코인으로 자동전환(AutoSwitch)할 때 `[FATAL] Cannot access a disposed object` 오류 발생하던 현상 수정.
  `Dispose()`에서 `_api`를 즉시 dispose하지 않고 루프 종료 시 자체 정리하도록 변경.

- **autoEntryScore 미적용 버그 수정**
  설정의 AutoEntryScore 값이 실제 진입 판단에 반영되지 않던 문제 수정.
  `AnalyzeKYJ`에서 `sig.Score >= _cfg.AutoEntryScore` 체크 추가.

### 차트 개선

- **Y축 패딩 버그 수정 (소수점 코인)**
  AGLD(0.28), ENSO(1.9) 등 1 미만 가격 코인에서 Y축 최소 패딩이 1로 고정되어 차트가 비정상적으로 늘어나던 현상 수정.

- **Y축 레이블 자릿수 자동 조정**
  가격 범위에 따라 소수점 자릿수를 자동 조정:
  - 10,000 이상: 정수 (BTC `98,000`)
  - 100 이상: 2자리 (`123.45`)
  - 1 이상: 4자리 (`1.2345`)
  - 0.01 이상: 5자리 (`0.37851`)
  - 이하: 6자리

### UI 개선

- **초기 심볼 표시 버그 수정**
  앱 시작 시 상단 좌측 가격창에 이전 코인(예: AGLD) 가격이 "BTC"로 표시되던 현상 수정.
  심볼 목록 로드 후 `SelectedSymbolDisplay` 즉시 갱신하도록 수정.

- **가격 표시 정밀도 전체 개선 (`PriceHelper.Format`)**
  차트 헤더 현재가, 포지션 진입가, 로그 메시지 등 전 화면에서 `:N2` 고정 포맷으로 인해 소수점 코인 가격이 `0.19`처럼 잘려 표시되던 문제 수정.
  가격 크기에 따른 동적 포맷 적용.

## v0.2 (2026-02-22)

### 초기 기능 구현

- KYJ 엔진: EMA 크로스 + 복합지표 스코어링 (RSI, MACD, ADX, 거래량, 1H/4H 추세)
- ConfirmSignal: 신호 감지 후 4회(40초) 가격 확인 필터
- 손익분기 SL: 수익 1.5% 이상 시 진입가로 SL 이동
- 자동전환(AutoSwitch): 스캐너가 준비도 높은 코인으로 자동 전환
- 실전/테스트 모드 분리, 테스트 상태 세션 간 복원
- 멀티심볼 스캐너, 상위 10개 코인 준비도 분석
- 실시간 차트 (캔들스틱 + EMA7/21/50 + 거래량)
- 배포 인스톨러 (Inno Setup)
